# AuthLift - Next.js Authentication Starter Kit

**A production-ready, reusable authentication starter template built with Next.js 16, Better Auth, MongoDB, Resend, and Arcjet.**

This repo is designed to be your **go-to authentication foundation** for every future project. Clone it, customize the UI, and ship secure apps in minutes instead of weeks.

## Tech Stack

- **Next.js 16** (App Router, Server Components)
- **Tailwind CSS** Hero UI components
- **TypeScript**
- **Better Auth**
- **MongoDB**

## Features

- **Email/Password Authentication** (signup & login)
- **Social Login** – Google
- **Email Verification on Signup** – Required before first login
- **Welcome Email** – Sent automatically after successful email verification
- **Password Reset Flow** – Forgot password → reset link → new password
- **Beautiful Email Templates** – Powered by Resend + React Email (Tailwind styled)
- **Rate Limiting & Security** – Arcjet integration (signup protection, bot detection, rate limiting, shield)
- **Protected Routes** – Example dashboard with session-based protection
